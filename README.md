## Prerequisites

- Node.js (v20.14.0 or higher)
- npm (v10.2.4 or higher)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/prathmbavge/IEEE_Leaderboard.git
cd IEEE_Leaderboard
```

2. Install dependencies:
```bash
npm install
```
